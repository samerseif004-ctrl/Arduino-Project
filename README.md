# Arduino Drawbridge System

## Overview
Designed and built an automated drawbridge system using an Arduino Uno, ultrasonic sensor, servo motor, and LEDs to simulate a real-world bridge control system.

## Project Goal
The goal of this project was to create a simple automated system that detects an approaching object, controls traffic lights, and raises or lowers a bridge using a servo motor.

## Components Used
- Arduino Uno
- Ultrasonic sensor
- Servo motor
- Red, yellow, and green LEDs
- Resistors
- Breadboard
- Jumper wires

## How It Works
The ultrasonic sensor measures the distance of an approaching object. When the object is detected within a set distance, the traffic light sequence changes, the bridge raises using the servo motor, and the system waits before returning to normal operation.

## Features
- Object detection using ultrasonic sensor
- Servo-controlled bridge movement
- Traffic light sequence
- Simple automated control logic

## Skills Demonstrated
- Arduino programming
- Basic C++
- Sensor integration
- Servo motor control
